theme_seaborn

Usage

theme_seaborn()

Arguments

Name Description
style Style of axis background. whitegrid | darkgrid | nogrid | ticks
context Intended context for resulting figures. notebook | talk | paper | poster
gridweight Width of the grid lines. extra heavy | heavy | medium | light | None

Examples

ggplot(aes('carat', 'price'), data=diamonds) + geom_point() + theme_seaborn()
ggplot(aes('carat', 'price'), data=diamonds) + geom_point() + theme_seaborn(style='darkgrid')
ggplot(aes('carat', 'price'), data=diamonds) + geom_point() + theme_seaborn(context='poster')
ggplot(aes('carat', 'price'), data=diamonds) + geom_point() + theme_seaborn(gridweight = 'extra heavy')
ggplot(aes('carat', 'price'), data=diamonds) + geom_point() + theme_seaborn(style = 'nogrid')
